## ⚡ 四段式运镜公式速查

**适用场景**：AI漫剧、动作打斗、变身特效

```
开场(0-3s)：低角度跟拍/慢推
→ 建立场景氛围，展示角色初始状态

发展(4-8s)：环绕摇镜快切特写
→ 展示动作细节，特效开始显现

高潮(9-12s)：仰拍拉远定格慢放
→ 情绪/特效爆发，最震撼的瞬间

收尾(13-15s)：缓推特写渐暗
→ 意境落版，定格最终形态
```

**实战示例**：
```
0-3秒：低角度慢推，剑客立于山崖，长发飘扬
4-8秒：环绕摇镜，周身金色灵力涌动，符文环绕
9-12秒：仰拍拉远，低喝"破！"，金光迸射定格慢放
13-15秒：特写渐显，凤凰战神形态显现，定格
```

## 🎨 色调宣言关键词扩充

**放在提示词开头，影响整体画面风格**

### 单一色调
- 金红暖色调 - 热血、激昂、战斗
- 紫黑冷色调 - 神秘、暗黑、魔法
- 冰蓝冷色调 - 冰冷、理性、科技
- 橙黄暖色调 - 温馨、怀旧、日落
- 青绿冷色调 - 清新、自然、治愈

### 对比色调
- 暖冷色调剧烈反差 - 冲突、对抗、反转
- 红蓝对比 - 正邪对立、阵营对抗
- 金黑对比 - 神圣与黑暗

### 特殊色调
- 黑白水墨 - 中国风、武侠、禅意
- 赛博朋克霓虹 - 科幻、未来、城市
- 复古胶片暖黄 - 怀旧、回忆、情怀
- 高饱和炫彩 - 时尚、MV、年轻

## 🗣️ 对白嵌入技法

### 真人短剧对白结构
```
3-5秒：角色A说："[台词]"，[表情：叹气/惊讶]
5-8秒：角色B回应："[台词]"，[动作描述]
```

### AI漫剧对白嵌入
```
9-12秒：角色低喝"破！"，[特效爆发]
女皇低吟"这冰蚀之力…竟已入骨"，[动作]
```

### 对白设计要点
- 简短有力，符合角色身份
- 情绪词配合：叹气、惊讶、笑中带泪、怒吼
- 配合时间轴节奏

## 🎵 逐拍声音设计

### 示例结构
```
0-3秒：环境音 + 轻柔配乐起
4-8秒：伴随冰晶炸裂的脆响与低温嘶鸣
9-12秒：雷鸣滚滚配合剑刃破空声
13-15秒：配乐渐弱，定格音效
```

### 常用音效描述
- 元素音：冰晶炸裂脆响、火焰爆发轰鸣、雷霆滚滚
- 动作音：剑刃破空、脚步声、衣袂飘动
- 环境音：海浪声、风声、雨声、城市噪音
- 特效音：粒子消散、能量涌动、法阵共鸣
