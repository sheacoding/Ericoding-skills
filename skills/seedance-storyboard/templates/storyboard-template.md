
## 模板八：真人短剧模板

```
【格式】9:16竖屏，真人短剧风格，15秒
【氛围】温馨/搞笑/感人/悬疑

0-3秒：中景，[场景建立]，素材图1中的主体人物（[角色身份]）[初始状态/动作]
3-5秒：特写，[角色A]说："[台词]"，[表情描述]
5-8秒：[镜头切换]，素材图2中的主体人物（[角色身份]）回应："[台词]"，[动作描述]
8-12秒：[冲突/转折/互动]，[情绪变化]，[关键动作]
12-15秒：[结局]，[情绪落点]，[氛围营造：暖光/冷光]

【声音】[背景音乐] + [环境音效] + 自然对话
【参考】素材图1 [角色1描述]，素材图2 [角色2描述]
```

**对白设计要点**：
- 台词简短有力，符合角色身份
- 情绪词配合表情：叹气/惊讶/笑中带泪/怒吼
- 环境音配合场景：门铃声/脚步声/碗筷碰撞

## 模板九：互动接龙模板

```
【第X集】接龙系列
【格式】9:16竖屏，[简洁文字动画/真人演绎]风格，10-15秒

0-2秒：[回顾/建立规则]，[上集内容回顾/规则说明]
2-4秒：[本集核心内容]，[文字/动作展示]
4-6秒：[结果呈现]，[完整答案/演绎结果]
6-8秒：[过渡/强化记忆]
8-10秒：[社交钩子]，底部出现"[引导文案：你能接下去吗？/继续接龙！]"

【声音】轻快背景音乐 + [互动音效]
【要求】保持与前集视觉风格一致
```

**系列策划要点**：
- 第一集：建立规则 + 示范 + 抛出问题
- 后续集：承接 + 新内容 + 再次引导
- 每集结尾：评论区互动钩子

## 模板十：P视频/视频编辑模板

```
基于@视频1进行编辑：

【继承】
- 继承原视频的主体人物
- 保持@视频1的所有运镜和动作节奏
- 保留原视频的[音乐/配乐/节奏]

【修改1】[具体修改项：服装/背景/道具]
描述：将[原内容]改成[新内容]

【修改2】[具体修改项]
描述：[详细描述]

【修改3】去掉/添加 [元素：水印/文字/特效]

【保留】所有面部表情和肢体动作/其他未提及部分
```

**常用编辑指令**：
```
【换装】将服装从[原色/款式]改成[新色/款式]
【换背景】保持主体人物，背景从[原场景]改成[新场景]
【去水印】去掉视频中的水印/文字
【换人物】将视频中的[角色]替换成素材图X中的主体人物
【局部修改】仅修改[时间段/位置]的[具体元素]
```

## 常用镜头语言词汇表

| 中文 | 英文 | 用法 |
|------|------|------|
| 推镜头 | push in / zoom in | 强调主体，营造紧张感 |
| 拉镜头 | pull out / zoom out | 展现场景，营造疏离感 |
| 摇镜头 | pan | 左右/上下展示环境 |
| 移镜头 | dolly/truck | 平行移动，跟随主体 |
| 跟镜头 | follow | 跟随主体移动 |
| 环绕镜头 | orbit/circle | 360度展示主体 |
| 升降镜头 | crane up/down | 从上往下或从下往上 |
| 希区柯克变焦 | dolly zoom | 背景压缩/拉伸效果 |
| 手持晃动 | handheld | 纪实感/紧张感 |
| 一镜到底 | one shot | 不换镜头，连续拍摄 |
