# 海报 / 排版设计 提示词模板

## 提示词结构公式

```
[layout style] poster design for [purpose/event], [main visual element],
[typography style] with text "[copy]", [color palette], [mood/atmosphere],
[design style reference], print-ready, high resolution
```

## 推荐参数

| 参数 | 值 |
|------|----|
| quality | hd |
| size | 1024x1792（竖版海报）或 1792x1024（横版） |
| style | vivid |

## 关键修饰词库

**版式风格：**
- `Swiss International Style` — 国际主义版式
- `Art Deco geometric` — 装饰艺术几何
- `brutalist typography` — 暴力排版
- `editorial magazine layout` — 杂志排版

**色彩：**
- `monochromatic with accent` — 单色调加强调色
- `duotone effect` — 双色调
- `pastel muted palette` — 马卡龙色系
- `high contrast black and white` — 高对比黑白

**质感：**
- `paper texture` / `risograph print` / `screen print aesthetic` / `letterpress`
