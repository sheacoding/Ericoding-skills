# 技术图表 / 信息图 提示词模板

## 提示词结构公式

```
[diagram type] for [system/concept], [components/nodes listed],
[connection/flow direction], [visual style], [color coding scheme],
[annotation detail level], [background], [resolution]
```

## 推荐参数

| 参数 | 值 |
|------|----|
| quality | hd |
| size | 1792x1024（横版图表）或 1024x1024（方形） |
| style | natural |

## 精选示例提示词

### 系统架构图
```
Clean technical architecture diagram for a microservices cloud system,
showing client layer (mobile and web), API Gateway in the middle tier,
5 microservices boxes (Auth, User, Payment, Notification, Analytics),
message queue connecting services, PostgreSQL and Redis databases at bottom,
arrows showing request flow direction, icons for each service type,
flat design with blue and gray color scheme, white background, engineering diagram quality
```

### 数据流程图
```
Professional data pipeline flow diagram, left to right layout, starting with
data sources (S3, PostgreSQL, Kafka stream), arrow into ETL processing layer,
data warehouse (Snowflake) in center, branching to BI tools (Looker, Tableau)
and ML platform (SageMaker), color-coded by layer (orange ingestion, blue
processing, green output), rounded rectangle nodes, clean sans-serif labels,
technical whitepaper style
```

### 时间线信息图
```
Horizontal timeline infographic showing the history of the internet from 1969 to 2024,
key milestones as circular nodes on a central line, alternating above and below layout,
each node has year label, event name, and brief description, icons representing
each era, gradient color from cool blue (1969) to warm orange (2024), modern flat
infographic style, magazine editorial quality
```

### 比较对比图
```
Side-by-side feature comparison infographic, three columns for "Basic" "Pro" and "Enterprise"
pricing tiers, rows showing 8 different features with checkmark and cross icons,
highlighted middle column for recommended tier, clean table layout with subtle
alternating row shading, blue header row, modern SaaS pricing page aesthetic,
white background, clear typography hierarchy
```

### 概念思维导图
```
Radial mind map diagram for "Artificial Intelligence" central topic, 6 main branches
(Machine Learning, NLP, Computer Vision, Robotics, Ethics, Applications), each
with 3-4 sub-nodes, connecting lines with varying thickness showing hierarchy,
different colors for each main branch, circular node shapes, clean white background,
MindNode app aesthetic, concise text labels, balanced radial layout
```

## 关键修饰词库

**图表类型：**
- `system architecture diagram` — 系统架构
- `entity relationship diagram (ERD)` — 数据库关系
- `flowchart with decision diamonds` — 流程图
- `Gantt chart project timeline` — 甘特图
- `treemap data visualization` — 树状图

**视觉风格：**
- `flat design icons` / `isometric 3D technical style`
- `whitepaper technical drawing` / `engineering blueprint`
- `infographic magazine style` / `minimal startup aesthetic`

**配色：**
- `blue and gray professional` / `warm editorial orange`
- `color-coded by category` / `traffic light red-yellow-green`
