# 人物 / 肖像 提示词模板

## 提示词结构公式

```
[lighting], [shot type] portrait of [subject], [clothing/accessories],
[background], [style/medium], [mood/expression], [quality modifiers]
```

## 推荐参数

| 参数 | 值 |
|------|----|
| quality | hd |
| size | 1024x1024（方形）或 1024x1792（竖屏） |
| style | natural（写实）/ vivid（艺术感） |

## 关键修饰词库

**光线：**
- `soft studio lighting` — 柔和棚拍
- `golden hour backlighting` — 黄金时刻逆光
- `dramatic Rembrandt lighting` — 伦勃朗戏剧光
- `rim light silhouette` — 轮廓光剪影
- `diffused overcast natural light` — 散射自然光

**风格：**
- `photorealistic, hyperdetailed` — 超写实
- `digital oil painting` — 数字油画
- `editorial photography` — 时尚杂志摄影
- `ink wash painting` — 水墨风格

**情绪：**
- `confident, assured` / `mysterious, introspective` / `warm, approachable`
- `serene and dignified` / `fierce and determined`
