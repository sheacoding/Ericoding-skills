# 产品渲染 提示词模板

## 提示词结构公式

```
[shot type] product photography of [product name/description],
[material/finish description], [background/surface], [lighting setup],
[styling props], [brand aesthetic], [quality modifiers]
```

## 推荐参数

| 参数 | 值 |
|------|----|
| quality | hd |
| size | 1024x1024（方形，适合电商）或 1792x1024（横版场景图） |
| style | vivid |

## 关键修饰词库

**材质描述：**
- `matte brushed aluminum` / `polished chrome` / `soft-touch matte rubber`
- `transparent crystal glass` / `warm walnut wood grain`
- `woven textile texture` / `glazed ceramic`

**布光方式：**
- `three-point studio lighting` — 三点布光
- `backlit with rim highlight` — 逆光+轮廓光
- `soft box diffused` — 柔光箱
- `dramatic directional side light` — 戏剧侧光

**背景：**
- `pure white seamless` / `gradient gray` / `concrete texture`
- `marble slab` / `linen fabric` / `floating in mid-air`
