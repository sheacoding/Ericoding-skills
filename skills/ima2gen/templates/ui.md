# UI / 界面 / 线框 提示词模板

## 提示词结构公式

```
[UI type] design for [app/product category], [screen/view name],
[design system/style], [key UI components], [color theme],
[device frame context], [design tool aesthetic]
```

## 推荐参数

| 参数 | 值 |
|------|----|
| quality | hd |
| size | 1024x1792（手机屏）或 1792x1024（桌面/平板） |
| style | natural |

## 关键修饰词库

**设计风格：**
- `glassmorphism` — 磨砂玻璃
- `neumorphism` — 新拟物化
- `flat design 2.0` — 扁平设计
- `material design 3` — Material You
- `Apple Human Interface Guidelines` — iOS 规范

**组件关键词：**
- `card layout with shadow` / `modal dialog overlay`
- `data table sortable columns` / `timeline feed`
- `onboarding flow screens` / `empty state illustration`

**质量词：**
- `pixel perfect` / `production ready` / `Figma component`
- `Dribbble design quality` / `Behance case study visual`
