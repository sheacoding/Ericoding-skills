# 动漫 / 插画 提示词模板

## 提示词结构公式

```
[anime style reference], [character description with outfit],
[scene/setting], [action/pose], [lighting and atmosphere],
[color palette], [art quality modifiers]
```

## 推荐参数

| 参数 | 值 |
|------|----|
| quality | hd |
| size | 1024x1792（竖屏角色图）或 1024x1024（场景方图） |
| style | vivid |

## 关键修饰词库

**风格引用：**
- `Studio Ghibli style` — 吉卜力
- `by Makoto Shinkai` — 新海诚
- `KyoAni quality` — 京都动画
- `manhwa style` — 韩漫
- `Pixar 3D animation style` — 皮克斯

**质量修饰：**
- `detailed linework` / `clean cell shading` / `soft painterly style`
- `vibrant saturated colors` / `delicate watercolor wash`
- `trending on ArtStation` / `featured on Pixiv`

**构图：**
- `dynamic action pose` / `serene contemplative pose`
- `low-angle heroic shot` / `bird's eye view`
